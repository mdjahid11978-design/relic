# 项目 / Projects

> 📭 **目录现在是空的，这是正常的。**
> 当你和 AI 一起做了一个比较大或持续的事情（比如"开发一个网站"、"写一本书"），AI 会在这里自动创建项目文档来跟踪进度。你也可以手动创建。

## 创建项目文件示例 / Minimal Example

```markdown
# 项目名称 / Project Name

## 状态 / Status
进行中 / 已完成 / 已暂停

## 简介 / Summary
一句话描述项目目标。

## 关键决策 / Key Decisions
- YYYY-MM-DD：决策内容

## 相关文件 / Related Files
- 路径/文件名

## 备注 / Notes
其他需要记录的信息。
```

义体根据对话判断某个任务是否构成"项目"，如果是则自动创建。
Agents auto-create project files when a task qualifies as a "project".

用户也可以手动创建。
Users can also create files manually.
